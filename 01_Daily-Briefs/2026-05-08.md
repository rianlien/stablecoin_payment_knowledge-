# Daily Brief: 2026-05-08

## 今日の要点

- AWS が Amazon Bedrock AgentCore Payments（プレビュー）を 5 月 7 日に発表。Coinbase の x402 と Stripe の Privy ウォレットを基盤とし、AI エージェントが HTTP 402 フローを通じて API・ウェブコンテンツ・MCP サーバーの料金を USDC で自律決済できるクラウドネイティブインフラが初登場。主要クラウドとして Google Cloud（Pay.sh/5 月 5 日）に続き AWS が x402 採用を表明したことで、x402 プロトコルの事実上の標準化が決定的になりつつある
- CLARITY Act のステーブルコイン利回り妥協案（Tillis-Alsobrooks）の最終テキストが 5 月 1 日に公表。銀行預金等価の利回りを禁止しつつ「活動連動型報酬（bona fide activities 基準）」を認める構造で、業界はすぐさま支持。上院銀行委員会のマークアップは 5 月中旬に計画され、Polymarket の法案通過確率が 60% 超に上昇。銀行業界 5 大団体は反発
- Kraken 親会社 Payward が香港のステーブルコイン決済企業 Reap Technologies を 6 億ドルで買収合意（5 月 7 日）。元 Stripe アジア太平洋責任者が創業した Reap のクロスボーダー B2B 決済インフラを取り込み、「取引所が決済レイヤーを丸ごと買収する」トレンドを鮮明化。規制承認（HK・シンガポール）待ち
- Solana Foundation × Google Cloud が x402 ベースの AI エージェント決済ゲートウェイ「Pay.sh」を 5 月 5 日に正式ローンチ。Gemini・BigQuery・Vertex AI 等を含む 75 以上の API に USDC on Solana での従量決済を提供。アカウント・API キー不要で AI エージェントが直接ペイ
- MoonPay が「MoonAgents Card」を 5 月 1 日に発表。Monavate（Mastercard プリンシパルメンバー）との共同開発で AI エージェントが USDC on Solana を使って Mastercard 加盟店で決済できる仮想デビットカード。UK・LATAM 先行提供

## 今日の重要論点

1. **x402 が「クラウド標準」に格上げされた週**: AWS（AgentCore Payments/May 7）・Google Cloud（Pay.sh/May 5）が相次いで x402 を採用したことで、x402 は「Coinbase 発の実験的プロトコル」から「メジャークラウドプロバイダーが採用する AI エージェント決済標準」に転換した。Stripe が MPP を推進しながらも x402 Foundation の創設メンバーでもある構図の中で、クラウド 2 社が x402 側についたことは MPP との競争において決定的な影響を持つ

2. **取引所が「決済インフラ企業」になる競争が本格化**: Kraken の Reap 買収（$600M）は、暗号資産取引所が単なる売買の場から「ステーブルコイン決済の垂直統合プロバイダー」へ転換する動きの象徴。Coinbase は Agentic.market と x402 で決済フロー参加、Stripe は Privy ウォレットと MPP で決済インフラを押さえ、Kraken は買収で決済企業を内製化する——三者三様のアプローチが競合する構図

3. **AI エージェントの「支払い能力」が二層構造に**: 今週の動向で、AI エージェントの支払い手段は「① x402 経由のオンチェーン API 従量決済（Pay.sh・AgentCore）」と「② カードネットワーク経由のオフチェーン加盟店決済（MoonAgents Card）」の 2 層に整備されつつあることが明確になった。用途に応じた使い分けが想定される

4. **CLARITY Act の利回り条項確定は PSP の報酬設計に直結**: 「bona fide activities 連動型報酬のみ許容」という枠組みが法制化されれば、PSP・PayFac が提供するステーブルコインキャッシュバック・報酬プログラムの設計が法的に確定する。7 月議会休会前の通過が現実的になってきた

## 共通して見える変化

- **x402 の採用が「プロジェクト → スタートアップ → メジャークラウド」と上流に拡張**: Coinbase（2025 年）→ Linux Foundation（2026-04）→ Google Cloud・AWS（2026-05）という採用の上流化が続いている。この流れは Stripe MPP が「自社エコシステム内での標準」にとどまる可能性を示唆する
- **エージェント決済のターゲット加盟店が「API」から「実世界加盟店」へ**: AWS AgentCore が「ホテル予約・旅行手配」への拡張を予告し、MoonAgents Card が Mastercard 加盟店をカバーすることで、AI エージェントが日常的な商取引全般で支払い主体となるインフラが整いつつある
- **規制（CLARITY Act）・インフラ（x402/クラウド統合）・M&A（Kraken/Reap）の三つが同時進行**: これらが揃うと、ステーブルコイン決済の主流金融インフラへの組み込みが一気に加速する可能性がある
