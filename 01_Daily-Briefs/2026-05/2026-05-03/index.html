    Daily Brief: 2026-05-03¶

    今日の要点¶

    • Walmart 傘下の OnePay（月間数百万ユーザー）が Tempo ブロックチェーンと提携し、ステーブルコイン送金・即時口座入金機能を追加すると発表。OnePay は Tempo バリデータも自社運用する。これで Tempo は DoorDash（4/21）・Coastal Bank（4/21）・Klarna（testnet 済み）・Visa（清算 4/29）・OnePay（4/29）という主要パートナーを 2 週間で揃えた
    • 韓国最大カード会社・新韓カード（2,800 万会員）が Solana Foundation と MoU を締結し、ステーブルコイン決済・ノンカストディアルウォレット・ハイブリッドファイナンスモデルの PoC を開始。4 月 9 日には 6 項目のブロックチェーン PoC 完了を既に公表済みで、今回はその次ステップとなる
    • Western Union の Digital Asset Network（DAN）が 4 月最終週に初パートナーとの稼働を開始。DAN は暗号資産ウォレットと WU の 36 万拠点を API 接続するオフランプインフラで、年内に 7 社以上のパートナー追加を予定。USDPT（Solana）は 5 月ローンチ予定を維持
    • Coinbase Asset Management が Superstate FundOS を採用し、ステーブルコイン担保クレジットファンド CUSHY のトークン化シェアクラスを Q2 2026 中に発行予定。Ethereum・Solana・Base の 3 チェーンに対応し、FundOS 初の外部ファンド事例となる

    今日の重要論点¶

    1. Tempo が 2 週間で主要プレイヤーを集結させた: DoorDash（4/21）・OnePay（4/29）・Visa 清算（4/29）・Klarna（testnet）に加え、Stripe 自身も Tempo を BridgeのバックエンドとしてSessions 2026で拡充した。Stripe が自社インフラとして Tempo を推進する構造は、Tempo を「Stripe の決済ブロックチェーン部門」として事実上機能させている。圧倒的なパートナー蓄積速度は、他のチェーン（Polygon・Avalanche 等）との差別化を急速に形成しつつある

    2. アジア太平洋圏の大手カード発行体が独自に動き始めた: 新韓カードの Solana MoU は、米国主導のインフラ（Stripe / Visa / Circle）に依存せず、既存カード網にブロックチェーン決済を直接統合しようとするアジア独自の動きを示す。韓国 FSC の規制対応が進めば、日本・シンガポール・台湾の同規模の発行体も追随しうる

    3. ステーブルコインのオフランプが「現金ネットワーク API」として本格稼働: Western Union DAN の初稼働は、ステーブルコイン決済の「最後の 1 マイル（現金化）」問題を世界最大規模のネットワーク経由で解決する最初の実用事例。オフランプ設計のデファクトが「取引所経由」から「現金ネットワーク API 経由」に移行し始めた可能性

    4. ステーブルコインが利回り資産化: Coinbase CUSHY は、ステーブルコインを「決済手段」として保有するだけでなく、「貸付利回り生成資産」として運用する機関ニーズに応える。PSP が自社のステーブルコイン残高管理に「利回り提供」を組み込む際の参照商品になりうる

    共通して見える変化¶

    • Tempo への集中が加速: 前週の DoorDash・Visa 拡張に続き、OnePay が加わったことでTempo は「Stripe エコシステム全体のステーブルコイン清算チェーン」としての地位を固めつつある。x402（Linux Foundation / Coinbase）との 2 軸構造がより鮮明に
    • 地理的拡大フェーズへ: これまでの米国・欧州中心から、韓国（新韓カード）・グローバル（WU DAN 36 万拠点）と、アジアおよび非銀行圏への展開が具体化し始めた
    • 「インフラ参画」型パートナーシップ: OnePay がバリデータを運用する設計は、パートナー企業が単なる API 接続を超えてチェーンのセキュリティ・ガバナンスに参画するモデルを示す。DoorDash・Coastal Bank も類似の深い統合を進めている可能性

    我々への示唆¶

    • いま検討すべきこと:
    • Tempo バリデータ運用がパートナー企業に与えるインセンティブ・コスト・法的位置づけを調査（自社の Tempo 統合深度を判断するため）
    • 新韓カードの PoC で使用するステーブルコインが USDC か韓国ウォン建てかで、自社の韓国市場戦略への示唆が変わる。公開情報を継続監視
    • WU DAN の API 仕様が公開された場合、自社オフランプ設計のオプションとして評価
    • 継続監視すべきこと:
    • OnePay の GA タイムラインと実際の対応ステーブルコイン種別
    • 韓国 FSC の仮想資産規制とカード発行体のステーブルコイン取り扱いに関するポジション変化
    • USDPT（Western Union）の 5 月ローンチ正式発表と対象国リスト
    • CUSHY Q2 ローンチ実績（利回り水準・投資条件）
    • 追加で深掘りしたいこと:
    • Tempo のバリデータ経済（報酬・スラッシング条件）と FInCEN/OCC 規制上の位置づけ
    • 新韓カードの PoC 設計（特に KYC/AML フローとオラクル選定）が公開された場合の詳細確認
