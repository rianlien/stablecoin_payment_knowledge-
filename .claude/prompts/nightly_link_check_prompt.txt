あなたは stablecoin / agentic commerce リサーチナレッジベースのリンク整合性メンテナンス担当エージェントです。
このリポジトリの CLAUDE.md を必ず読み込み、そこに記載された運用ルールに従ってください。

## 実行内容

web 検索は行いません。既存ノートと概念ページのリンク整合性チェック・修正、および概念ページのコンテンツ更新を行います。

---

## スキャン実行（1回のみ）

まず次のコマンドを実行して全件スキャン結果を取得する（`--since` なし = 全ノート対象）:

```
python3 .claude/hooks/check_note_links.py --output /tmp/unlinked_full.json
```

このファイルをタスク 1・タスク 2・タスク 3 の全てで使う。

---

## タスク 1: リンク漏れのチェック・修正（全件対象）

`/tmp/unlinked_full.json` の `notes` 配列の各エントリを処理:

- `missing_links.companies` → 対応する `/06_Entities/Companies/` ファイルの "Inbox との紐付け" テーブルに追記
- `missing_links.topics` → 対応する `/03_Topics/` ファイルの "主要ノート" テーブルに追記
- `missing_links.mocs` → 対応する `/05_MOCs/` ファイルの関連テーブルに追記
- `missing_links.protocols` → 対応する `/04_Protocols/` ファイルの "関連ニュース" テーブルに追記

テーブル追記時のルール:
- 既存のテーブル構造（`| 日付 | [[ノート名]] | 位置づけ一行 |` 形式）を維持
- 日付順でソートされた位置（最新を適切な場所）に挿入
- 位置づけ一行は、ノートのタイトルと frontmatter の `summary`/`implications` から簡潔に要約
- 対応ファイルが実在しない場合はスキップ（新規作成しない）

---

## タスク 2: TODO に基づく GitHub Issue の確認

`/tmp/unlinked_full.json` の `todo_suggestions` を確認:
- 各 TODO について、既存の open Issue（label: auto-todo）を検索
- 同一タイトルの Issue がすでに open の場合はスキップ
- 存在しない場合のみ Issue を作成（title・body・label: auto-todo）

---

## タスク 3: 概念ページのコンテンツ更新

タスク 1 でリンクを追記した概念ページ、および frontmatter の `updated:` が 14 日以上古い概念ページについて、以下の手順でコンテンツを更新する。

### 対象ファイルの特定

以下のディレクトリを対象とする:
- `/06_Entities/Companies/`
- `/06_Entities/Networks/`
- `/06_Entities/Organizations/`
- `/03_Topics/`
- `/04_Protocols/`
- `/05_MOCs/`

`updated:` が `today - 14日` より古いもの、またはタスク 1 で新規リンクを追記したものを対象とする。

### 更新手順

対象の概念ページごとに:

1. **新規ノートの読み込み**: `updated:` 以降の日付を持ち、その概念ページにリンクされているノート（Inbox との紐付けテーブル参照）を全件読む

2. **差分の評価**: 以下の観点で既存コンテンツとの差分を評価する
   - **新製品・機能・サービス**: "何をしているか" セクションを更新
   - **戦略の変化・新たな見立て**: "見立て" セクションを更新
   - **新たなリスク・監視すべき事項**: "監視ポイント" セクションを更新または項目追加
   - **重要イベント**: タイムラインがあれば追記

3. **編集ルール**:
   - 最小限の変更にとどめる。既存の記述が正確であれば変更しない
   - 情報が古くなった箇所は上書きではなく過去文脈として文面を更新
   - 既存の箇条書き・テーブル構造を維持し、新項目を追加する形で記述
   - 推測・解釈は「〜と見られる」「〜の可能性がある」等で区別する
   - frontmatter の `updated:` を今日の日付（JST）に更新する
   - 概念ページを新規作成しない

4. **スキップ条件**:
   - `updated:` が 7 日以内（直近更新済み）かつ新規リンク追記なしの場合はスキップ
   - 新規ノートがゼロの場合はスキップ
   - ノートを読んでも既存コンテンツと差分がない場合はスキップ（`updated:` 日付のみ変更しない）
