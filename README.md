## Purpose
ステーブルコイン決済関連ニュースと示唆を、チームで再利用可能な Markdown 資産として蓄積する。

## Folder rules
- 00_Inbox: 1ニュース1ファイル
- 01_Daily-Briefs: 日次まとめ
- 02_Weekly: 週次まとめ
- 03_Topics: MOCや論点整理
- 90_Templates: テンプレート

- 出典を残す
- 既存タグを優先する
- 規制ニュースは誇張しない
