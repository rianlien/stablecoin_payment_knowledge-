    Celo、Stripe 傘下 Bridge に対応——単一 API で USDC の Celo オンランプ・オフランプ・クロスチェーン送金が可能に。66 か国以上で決済レールとして稼働中の Celo がグローバル PSP インフラに接続

    概要¶

    2026 年 5 月 12 日、CoinDesk 主催 Consensus Miami 2026 の Day 2 で、Stripe が 2024 年に買収したステーブルコインオーケストレーションプラットフォーム Bridge が Celo ブロックチェーンへの正式対応を発表した。Bridge のシングル API を通じて、Celo 上での USDC オンランプ・オフランプ・クロスチェーン移動が可能になり、Bridge を利用する世界中の開発者・PSP・フィンテック企業が Celo のステーブルコインエコシステムへアクセスできるようになった。

    何が起きたか¶

    • 発表日：2026 年 5 月 12 日（Consensus Miami Day 2）
    • underlying_event_date：2026-05-12
    • 発表者：Bridge（Stripe 傘下）

    Bridge × Celo 統合の内容¶

    • 機能：Bridge のシングル API 経由で Celo 上の USDC について以下が可能に：
    • オンランプ：法定通貨 → Celo 上 USDC への変換
    • オフランプ：Celo 上 USDC → 法定通貨への変換
    • クロスチェーン転送：他チェーン（Ethereum・Solana 等）との間で Celo 上 USDC を移動
    • 従来の課題の解消：Bridge は「コルレス銀行ナイトメア」（多段階の銀行仲介）を回避し、66 か国以上で直接決済を実現するとしている

    Celo ネットワークの現状（2026 年 5 月時点）¶

    指標 値
    ライフタイムトランザクション 1,300 億回超
    ステーブルコイン累計取引量 650 億ドル超（2025 年 3 月 ETH L2 移行後）
    日次アクティブユーザー 60 万人超
    ネイティブステーブルコイン種類 25 種類
    稼働国 66 か国以上
    • 2025 年 3 月に Ethereum L2 に移行（EVM 完全互換）
    • アフリカ・東南アジア・ラテンアメリカでモバイルファースト型のデジタルドルレールとして普及

    なぜ重要か¶

    決済事業者視点¶

    • Bridge API を既に利用している PSP は、追加統合なしで Celo レール経由の新興国向けオフランプを追加できる。アフリカ（特にケニア・ガーナ・南アフリカ）・ラテンアメリカ・東南アジアへの送金・決済ルートが拡充される
    • Stripe / Bridge が Celo を採用したことは、Celo が「スタートアップ向け決済チェーン」から「グローバル PSP インフラ」として認められた証左。将来的に Solana・Ethereum と並ぶ主要な B2B 決済チェーンとして位置付けられる可能性がある

    加盟店視点¶

    • 66 か国以上のコルレス銀行不要の直接決済という Celo の強みが、Bridge 経由で西側フィンテックツールを使う加盟店にも届くようになった。特に新興国向けのクロスボーダー B2B 決済・サプライヤー支払いで実用的なオプションが増える

    プロダクト視点¶

    • Celo には 25 種類のネイティブステーブルコインがあり、地域通貨建て（例：cKES＝ケニアシリング建て）も含まれる。Bridge 対応後、これらの地域通貨建てステーブルコインへのアクセスが単一 API で可能になれば、現地通貨建て決済フローの設計が大幅に簡素化される

    規制 / リスク視点¶

    • Celo は Ethereum L2 として EVM 完全互換であるため、既存の Ethereum ベースのコンプライアンスインフラ（KYT・ブロックチェーン分析ツール等）がそのまま適用可能
    • Bridge（Stripe 傘下）が対応したことで、Celo レールの機関・PSP 向け採用における信頼性が向上した

    我々への示唆¶

    • いま検討すべきこと:
    • 自社が Bridge API を利用している場合、Celo オフランプの追加コスト・対応通貨・到達国リストを Bridge に問い合わせ、新興国向けオフランプ設計の改善に活用できるかを評価する
    • Celo 上の地域通貨建てステーブルコイン（cKES 等）への対応可能性を調査し、アフリカ・東南アジア向けサービスがある場合の現地通貨決済オプションとして検討する
    • 後で深掘りすべきこと:
    • Bridge の Celo 対応の具体的な対応通貨・対応国のリスト
    • Celo の 25 種類ネイティブステーブルコインのうち、地域通貨建て（cREAL・cKES・cCOP 等）の決済事業者向け利用可能性
    • 今は優先度が低いこと:
    • Celo のトークノミクス・バリデーター構造（決済事業への直接影響が限定的）

    未解決の論点¶

    • Bridge の Celo 対応は USDC のみか、Celo 上の 25 種類のネイティブステーブルコイン全体に及ぶか？
    • Bridge 経由の Celo オフランプの手数料・速度・対応通貨ペアは？
    • Celo と Solana・Ethereum でオフランプ手数料・速度にどれほどの差があるか？
