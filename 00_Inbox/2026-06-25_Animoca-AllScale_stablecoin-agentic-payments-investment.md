## 概要

- Animoca Brands が stablecoin-native 決済スタック企業 AllScale に戦略投資（金額非公開、2026-06-25 発表）
- AllScale：checkout・payroll・invoicing・pay-in/pay-out を統合した end-to-end ステーブルコイン決済プラットフォーム
- 登録ウォレット 1.5M+ 、クロスボーダー決済・非カストディアル決済に特化
- Animoca の 600 社超ポートフォリオでのグローバル決済・treasury・agentic commerce ユースケース展開を協力して探索予定

## 何が起きたか

- Animoca Brands が AllScale への戦略投資を 2026-06-25 に公式発表
- AllScale がこの投資を自社サイトで同日確認
- 両社は「グローバル決済・決済・treasury サービス、および agentic commerce（事前定義された上限内で規制準拠ステーブルコインを決済レールとして使用する AI エージェント決済）」の展開を探索予定と発表

## 確認された事実

- 投資: Animoca Brands による戦略投資（金額・ラウンドサイズは非公開）
- 発表日: 2026-06-25
- AllScale の現状規模: 1.5M+ 登録ウォレット
- AllScale のサービス範囲: checkout / payroll / invoicing / pay-in / pay-out、クロスボーダー決済
- AllScale の設計: 非カストディアル、マルチチェーン資産ブリッジ・スワップ、トランザクションスクリーニング・オンチェーンプライバシー機能内包
- Animoca Brands のポートフォリオ: 600 社超

## なぜ重要か

### 決済事業者視点

- AllScale の「stablecoin-native × 非カストディアル × マルチチェーン」設計は、PSP が新興市場・Web3 ゲーム・アジア太平洋向けステーブルコイン決済レールを構築する参照例として有用
- Animoca の 600 社超ポートフォリオ（NFT・ゲーム・メタバース）は agentic commerce の先行ユースケースの宝庫——どのステーブルコイン決済パターンが実際に機能するかを早期に観察できる

### 加盟店視点

- Animoca ポートフォリオ企業が AllScale を経由してステーブルコイン決済・payroll・invoicing を実装すれば、Web3 ゲーム・メタバース向け B2B ステーブルコイン決済のリファレンスモデルが出現する
- 加盟店が USDC/USDT などのステーブルコインで受け取り → 自国通貨で精算できる仕組みを提供（Crypto 非対応加盟店も利用可能）

### プロダクト視点

- AllScale の「predefined limits 内のエージェント決済」設計は、x402 の HTTP 402 フロー・Mastercard AP4M の事前承認モデルと概念的に親和性がある
- AllScale が x402 または MCP ツール対応を発表すれば、エージェント決済プロトコルの実装例が 1 件追加される

### 規制 / リスク視点

- AllScale は「regulated stablecoins as settlement layer」を強調しており、GENIUS Act 準拠を意識した設計
- 非カストディアル設計はカストディリスクを分散させるが、ML/TF リスクスクリーニング責任の所在が明確か要確認

## 我々への示唆

- いま検討すべきこと:
  - AllScale の API 仕様（checkout / payroll / invoicing）と x402 / MCP との統合可能性を技術評価。1.5M ウォレット規模のネットワークが使えるかを確認
  - Animoca ポートフォリオ企業向けの agentic commerce 決済ユースケースが登場した場合の参入機会を検討
- 後で深掘りすべきこと:
  - AllScale の x402 対応ロードマップの有無・MCP ツール統合計画
  - Animoca のポートフォリオ企業（600 社超）が AllScale を通じてどの規制準拠ステーブルコインを主に使用するか（USDC vs USDT vs 独自）
- 今は優先度が低いこと:
  - AllScale の過去の資金調達履歴（$1.5M 初期ラウンド）——今回は戦略投資のため規模より戦略的意図が重要

## ありそうな示唆

- Animoca の Web3 ゲーム・メタバース企業 600 社超ポートフォリオ全体でステーブルコイン決済が標準化されれば、デジタル財・ゲームアセット取引のステーブルコイン決済量が急増する可能性
- Yat Siu の発言「regulated stablecoins are the ideal bridge between traditional and on-chain financial systems, and are perfectly positioned to serve as core payment rails for emerging agentic commerce」は、Animoca の戦略的方向性として重要なシグナル
