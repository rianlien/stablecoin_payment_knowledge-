## 概要

2026 年 6 月 22 日、MoneyGram が Solana のアクティブバリデーターとして参加するとともに、機関向け Solana Developer Platform への参加を発表した。バリデーター運営では SOL をステークし、トランザクションブロックの処理とネットワークセキュリティに貢献する。Solana は MoneyGram が公式バリデーターを運営する 3 つ目のブロックチェーン（Tempo・Midnight Network に続く）。発表の 20 日前（6/2）には Stellar 上で自社ステーブルコイン MGUSD を本番ローンチしており、2 チェーン体制への移行が鮮明になった。

## 何が起きたか

- MoneyGram が PR Newswire でプレスリリースを発行（2026-06-22）
- Solana バリデーターとして SOL ステーキング・ブロック処理・ネットワークセキュリティ貢献を開始
- Solana Developer Platform に参加——Mastercard 等の機関と並ぶ早期採用者（early adopter）として参加
- Solana Developer Platform の概要：「AI-ready、API-driven」なコンプライアント金融製品の設計・構築・スケール基盤
- MoneyGram のバリデーター履歴：Tempo（Stripe 傘下）・Midnight Network（Cardano エコシステム）に続く 3 例目
- 直前の背景：MGUSD ステーブルコインを Stellar で 2026-06-02 に本番ローンチ（[[2026-06-02_MoneyGram_mgusd-stellar-launch]]）

## 確認された事実

- PR Newswire 公式プレスリリースで確認（2026-06-22）
- CoinDesk・The Block・Genfinity 等が同日報道
- MoneyGram の規模：60 万顧客（60M）、約 200 カ国・50 万リテール拠点
- Solana は第 3 のバリデーター運営チェーン（Tempo・Midnight Network に続く）
- Solana Developer Platform には Mastercard も参加

## なぜ重要か

### 決済事業者視点

- MoneyGram が Solana バリデーターとして参加することで、Solana チェーン上の取引処理に直接関与。ステーキング収益（SOL 報酬）という新たな収益源を得ると同時に、Solana 上のステーブルコイン決済インフラ（特に USDC・Solana Pay）の「動く事業者側」に立場が変わる
- MGUSD の Stellar 専用設計が Solana にも拡張される布石と読める

### 加盟店視点

- MoneyGram が Solana エコシステムに深く統合されることで、Solana 上の x402・Solana Pay 経由の決済 → MoneyGram ネットワーク経由のオフランプ（現金化・銀行振込）というフローが将来的に整備される可能性がある
- 60M 顧客を持つ MoneyGram のオフランプ網が Solana 上のステーブルコイン決済に接続されれば、AI エージェント決済 → 現地通貨受け取り（グローバル新興国含む）のルートが開ける

### プロダクト視点

- Solana Developer Platform は「AI-ready, API-driven」と表現されており、エージェント決済・機械間決済向けの API 開発ツールを提供する基盤。MoneyGram がここに参加することで、Solana 上での AI エージェント向け MoneyGram API の構築が可能になる
- MGUSD が Solana に対応した場合、MGUSD + x402 + Solana Pay の組み合わせで、グローバルの MoneyGram ネットワーク向けエージェント決済フローが成立する

### 規制 / リスク視点

- Solana バリデーター運営にはステーキングリスク（スラッシング等）があるが、MoneyGram 規模の機関に対してこのリスクは小さい
- Midnight Network（Cardano の privacy chain）バリデーターも兼任していることから、MoneyGram は「プライバシー保護決済」への関心を示している
