## 概要

- Circle が 2026 年 5 月 11 日、AI エージェント向けインフラ製品群「Circle Agent Stack」を正式発表・提供開始した。
- 同日、Circle は Arc ブロックチェーンの ARC トークンプレセールを $222M（評価額 $3B FDV）で完了したと発表。BlackRock・a16z・Apollo・ICE（NYSE 親会社）・SBI・General Catalyst 等が参加。
- Circle は「ステーブルコイン発行体」から「エージェント経済を動かすインフラ」へのポジション転換を、IPO 後初の大型発表として内外に明示した形。
- Q1 2026 業績：売上高 $694M（前年比 +20%）、USDC 流通量 $77B、オンチェーン取引量 $21.5T。

## 何が起きたか

**Circle Agent Stack（2026 年 5 月 11 日ローンチ）**

初期製品ラインナップ：
1. **Circle CLI**：エージェント開発者向けコマンドラインツール
2. **Agent Wallets**：AI エージェントがプログラマブルに USDC を保持・送金できるウォレット
3. **Agent Marketplace**：エージェントがサービスを発見・取引できるマーケットプレイス
4. **Nanopayments（powered by Circle Gateway）**：ガスフリーの USDC マイクロペイメント（最小単位：$0.000001 ≒ 100 万分の 1 ドル）

**Arc ブロックチェーン + ARC トークンプレセール（同日発表）**

- Arc は機関向け公開ブロックチェーン。ガストークンとして USDC を使用し、機関に予測可能なドル建てコストを提供。
- ユースケース：ステーブルコイン・トークン化資産・経済コントラクト・オンチェーンマーケット・AI 主導の決済活動。
- テストネット稼働中（2025 年 10 月〜）、メインネット公開は 2026 年夏予定。
- プレセール $222M 完了。評価額 $3B FDV。a16z crypto（リード・$75M）・BlackRock・Apollo・ICE・SBI・Janus Henderson・Standard Chartered Ventures・General Catalyst・Marshall Wace・ARK Invest・IDG Capital・Haun Ventures・Bullish が参加。
- Circle は上場企業として初のトークンプレセール実施（業界初の事例として注目）。

## 確認された事実

- 発表日：2026 年 5 月 11 日（BusinessWire / CNBC / CoinDesk / The Block）
- Circle 株（NYSE: CRCL）は同日 +15.9% 上昇
- ARC プレセール：$222M、FDV $3B、a16z がリード（$75M）
- Arc アーキテクチャ：USDC をガストークンとして使用（変動する native token ではなく）
- Circle CEO Jeremy Allaire：「Arc は単なるステーブルコインや決済ではなく、実経済を動かすことができる」と表明
- Q1 2026：売上高 $694M（+20% YoY）、USDC 流通量 $77B、オンチェーン取引量 $21.5T
- 同日 Consensus Miami 2026 で発表（同会場での業界最大規模のアナウンスの一つ）

## なぜ重要か

### 決済事業者視点

- Circle が自社ブロックチェーン（Arc）＋エージェントウォレット＋マーケットプレイスを構築したことで、「USDC を使って決済する」ための全レイヤーを Circle が単独で提供できる体制が整い始めた。
- Circle が PSP・エージェント決済プラットフォームに対して「パートナーとして決済インフラを提供する」から「直接エンドユーザーに提供する垂直統合プレイヤー」に移行する可能性がある。
- Arc vs Tempo（Stripe + Paradigm）：どちらも USDC を中心に据えた決済特化 L1 として競合する構図。

### 加盟店視点

- Agent Marketplace が実用化されれば、エージェントが Circle インフラ内でサービス発見→決済→受け取りまで完結するエコシステムが形成される。加盟店は Circle エコシステムへの参加が競争上重要になりうる。

### プロダクト視点

- Nanopayments（$0.000001 最小単位）は AI トークン消費・API コール課金・IoT マイクロペイメントに対応する。Stripe × Tempo のストリーミング決済（5 月 13 日ノート）と同様の「AI 利用量課金」ユースケースを狙う。
- Agent Wallets が複数チェーンに対応するかどうか（現時点未確認）で、Circle Agent Stack の汎用性が決まる。

### 規制 / リスク視点

- 上場企業初のトークンプレセールとして、SEC の審査対象になる可能性がある。Howey テスト・CLARITY Act の適用範囲が焦点。
- BlackRock が ARC トークンプレセールに参加したことは、機関投資家がブロックチェーン特化トークンへの直接投資を開始したことを示す業界マイルストーン。

## 我々への示唆

- いま検討すべきこと:
  - Circle Agent Stack の API 仕様（Agent Wallets の対応チェーン・Nanopayments の精算設計）を確認し、Stripe × Tempo・Coinbase x402 との比較評価を開始する。
  - Arc と Tempo の技術・ポジショニング差異を整理し、エージェント決済インフラとして自社が乗るべきレールを評価する。
- 後で深掘りすべきこと:
  - Arc メインネット公開（2026 年夏予定）後の API・手数料・パフォーマンス詳細。
  - Circle Agent Stack と Coinbase AgentKit・MoonPay AgentKit の機能差と統合可能性。
  - ARC トークンの規制上の取り扱い（SEC 審査状況）と CLARITY Act 適用の有無。
- 今は優先度が低いこと:
  - ARC トークン投資の判断。決済インフラとしての評価を優先する。

## ありそうな示唆

- Circle の動きは「USDC 依存型の他社エージェント決済インフラ」を飲み込む可能性がある。Agent Wallets・Marketplace を持つ Circle は、エージェント決済の「デフォルトレール」として位置づく戦略を持っている。
- Stripe（Tempo 経由）と Circle（Arc 経由）のどちらが「AI エージェント決済の標準インフラ」になるかは、2026〜2027 年の最重要の産業競争の一つになりつつある。

## 推測 / 監視ポイント

- Arc メインネット稼働後の実取引量・採用 dApps の規模
- Agent Wallets の対応チェーン（Arc 専用か、Solana・Ethereum 等マルチチェーンか）
- SEC が Circle の ARC トークンプレセールをどう取り扱うか（CLARITY Act 立法前の先行事例として注目）
- Stripe/Tempo との明示的な競合・協業ポジションの確定

## 未解決の論点

- Arc のメインネット TPS・最終確定性・クロスチェーン設計の詳細
- Circle Agent Marketplace の採用基準・手数料・KYC 要件
- Nanopayments の「Circle Gateway」と既存の Nanopayments Mainnet（4 月 29 日）との製品的な関係・統合状況
- ARC トークンのユーティリティ（Arc バリデーター報酬・ガバナンス・手数料割引 等）の詳細
