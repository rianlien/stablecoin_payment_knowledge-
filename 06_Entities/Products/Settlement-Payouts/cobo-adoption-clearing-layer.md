# Cobo Adoption Clearing Layer

## Overview

Cobo Adoption Clearing Layer は、public chains、stablecoin issuers、payment institutions を incentives 付きで結ぶ institutional stablecoin payment network 構想。

## User Goal

chain や stablecoin の採用を、実際の institutional payment flow に接続したい。

## Business Goal

Cobo が wallet infra 提供者から payment clearing coordinator へ役割を広げること。

## Category

compliance, checkout

## Core Workflow

ecosystem partner は ACL に参加し、payment institution 側の transaction demand と chain / stablecoin 側 incentive をつなぐ。実取引に応じたインセンティブでネットワーク利用を促進する。

## Pricing / Business Model

network / incentive layer 型。最大 15bps incentives が示されている。

## Differentiation

merchant checkout API ではなく、institutional payment adoption を clearing と incentives の問題として product 化している点。

## Operational Notes

導入前に screening responsibility、settlement data visibility、merchant / PSP surface、incentive durability を確認したい。

## Risks / Open Questions

- incentives が終わった後の持続性が不明
- end-merchant から見た usable surface が見えにくい
- network participant 間の責任分界が複雑になりやすい
