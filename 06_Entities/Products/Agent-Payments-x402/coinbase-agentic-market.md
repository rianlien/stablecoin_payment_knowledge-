# Coinbase Agentic.Market

## Overview

Agentic.Market は、x402 対応サービスの discovery marketplace。価格、出来高、カテゴリ、integration guide を一覧化し、人と agent の両方が探索できる。

## User Goal

使える paid agent service をすばやく見つけ、比較し、接続したい。

## Business Goal

x402 ecosystem の discovery layer を Coinbase 側で押さえ、service discovery のデフォルト面を取ること。

## Category

x402, agent-payments

## Core Workflow

利用者は marketplace で service を検索し、価格や signal を比較し、guide や bazaar 経由で integration する。

## Pricing / Business Model

明示価格は source 上で未確認。戦略的には x402 ecosystem の流通面・指名面を握る意味合いが強い。

## Differentiation

単なる directory ではなく、live pricing、volume、top list、guide を束ねている点。agent runtime 側からも使える構造。

## Operational Notes

discovery layer は trust signal の定義が重要。掲載面が強くなるほど、quality ranking と stale listing 管理が競争力になる。

## Risks / Open Questions

- ranking / trust signal の算出方法が未確認
- compliance-sensitive service の表示方針が未確認
- marketplace 内 conversion funnel が見えない
