# MoneyGram

MoneyGram は世界最大規模の送金事業者のひとつ。60M 顧客・約 200 カ国・約 50 万リテール拠点のグローバルネットワークを持ち、2026 年に自社ステーブルコイン MGUSD を Stellar でローンチ、Solana バリデーターへの参加を経て、マルチチェーン決済インフラ企業への転換を進めている。

---

## 現状の要約

### 何をしているか

- **MGUSD ステーブルコインのローンチ（2026-06-02）**: Stellar ブロックチェーン上で USD ペッグの自社ステーブルコイン MGUSD を本番ローンチ。発行体（PPSI）は Stripe 傘下の Bridge、スマートコントラクトは M0 Foundation、ウォレットインフラは Fireblocks が担当。MoneyGram アプリに自己カストディウォレットを組み込み、60M 顧客向けの「基盤通貨」として展開予定
- **Solana バリデーター参加（2026-06-22）**: Solana のアクティブバリデーターとして参加を発表。Solana は MoneyGram が運営する 3 つ目のバリデーターチェーン（Tempo・Midnight Network に続く）。あわせて機関向け Solana Developer Platform（AI-ready、API-driven）にも参加

### 見立て

MoneyGram は「送金事業者」から「マルチチェーン決済インフラ運営者」へのポジション転換を進めていると見られる。MGUSD（Stellar）＋ Solana バリデーター参加という 2 チェーン戦略が明確になりつつあり、Solana Developer Platform 経由で AI エージェント向け API の構築が将来的に見込まれる。MGUSD の Solana 対応が次のマイルストーンとなる可能性が高い。
